NAMING CONVENTION
	class name is written as oClass | oClassName
	::STATIC
		method → Method() | MethodName()
		property → $Property | $PropertyName
	$this->INSTANCE
		method → method() | methodName()
		property → $property | $propertyName
	ARGUMENTS
		$argument | $next_argument
	GENERIC
		$res ~ result, resolve
		$tsk ~ task
		$o ~ output
		$i

EXAMPLE
oUser::FirstName($user_id);
$oUser->firstName($user_id);
